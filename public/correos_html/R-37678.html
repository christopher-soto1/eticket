FortiGate Cloud 25.3 Release Highlights

Dear Customer,

We are pleased to announce that FortiGate Cloud v25.3 is scheduled for release on 2025-07-30. This update introduces several new features designed to enhance your experience and strengthen security:


  1. FortiGate Fabric Device Support
    • Display FortiGate connected fabric devices (FortiAP, FortiExtender, and FortiSwitch)
    • Manage fabric devices authorization/deauthorization and firmware upgrades
  2. Sandbox
    • Support downloading Sandbox Scan reports
    • FortiGuard Encyclopedia references are included in Sandbox Scan reports
  3. OU Dashboard
    • Support for CLI script tasks
    • Integration of Sandbox features
    • Support for administration and user settings
  4. ZTNA Support
    • ZTNA monitoring dashboard
    • ZTNA FortiView and drilldown
  5. Portal Enhancements
    • Enhanced report template editing
    • Firmware upgrade history now viewable by users
    • Reports can now be generated in CSV format

Thank you for choosing Fortinet. We look forward to providing you with these exciting improvements.

Sincerely,
The FortiGate Cloud Team
