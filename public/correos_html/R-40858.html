Junto con saludar, informo que por la nueva normativa de la Subtel se contrataron números con prefijos 600 y 809.

La ley 600 y 809 se refiere a la nueva normativa chilena que implementa prefijos telefónicos para diferenciar entre llamadas comerciales solicitadas y no solicitadas. El prefijo 600 se utiliza para llamadas que el usuario ha solicitado o autorizado, mientras que el 809 se usa para llamadas comerciales no solicitadas, como campañas publicitarias o de marketing. Esta medida busca proteger a los usuarios de llamadas spam y facilitar la identificación de llamadas legítimas de empresas con las que tienen relación, como bancos o aseguradoras. 

Dado todo esto y para no incurrir en una falta, se implementaron estas numeraciones.
